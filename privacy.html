Privacy and Policy

Helluva Bot 2.0's Discord support server https://discord.gg/xxk4MjMjUq

Data Stored

The following data is stored in our Helluva Bot 2.0's database

  • User's id (Snowflake) used to return a users game data.
  • Server's id (Snowflake) used to store a user's game data in that server.
  • Image links, inserting new characters into a server will store a link of their image in the database. The following data is stored temporarily in RAM.
  • Message id and content is stored temporarily in RAM when using certain game commands including /kdm, /smashpass, /guess , /gift-collectable, /waifu-trade, /collect-trade but the message contents and ids are stored no longer then 10 minutes or until the game requested is completed.

User and servers ids stored in the database are not shared with other servers or any third-party. The Helluva Bot 2.0 owner does NOT sell any information stored in the database.

Data Removal Request

Users can ask their data to be deleted from the database. This can be done by joining our support server and contacting user with the ID 673992089733955646 they will will respond to this request as soon as possible.

Users data can also be deleted by leaving the server the Helluva Bot 2.0 is located in or kicking the Helluva Bot 2.0 from the server.

Users present in a server with Helluva Bot 2.0 will not have their data collected by default unless they interect with bot or another user interacts with you using Helluva Bot 2.0.
